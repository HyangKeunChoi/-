## 다형성을 사용함으로써 갖는 장점

1. 다양한 여러 클래스를 하나의 자료형(상위 클래스)으로 선언하거나 형변환하여 각 클래스가 동일한 
메서드를 오버라이딩 한 경우, 하나의 코드가 다양한 구현을 실행 할 수 있음

2. 유사한 클래스가 추가되는 경우 유지보수에 용이하고
각 자료형 마다 다른 메서드를 호출하지 않으므로 코드에서 ***많은 if문이 사라짐***
