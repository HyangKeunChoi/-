## 형 변환

서로 다른 자료형의 값이 대입되는 경우 형 변환이 일어남

묵시적 형 변환 : 작은 수에서 큰수로, 덜 정밀한 수에서 더 정밀한 수로 대입되는 경우

명시적 형 변환 : 변환되는 자료 형을 명시 자료의 손실이 발생 할 수 있음
