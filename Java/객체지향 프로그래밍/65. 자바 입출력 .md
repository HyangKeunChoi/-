
# 자바 입출력 스트림

> 입출력 스트림이란?

네트워크에서 자료의 흐름이 물과 같다는 의미에서 유래

다양한 입출력 장치에 독립적으로 일관성 있는 입출력 방식 제공

입출력이 구현 되는 곳에서는 모두 I/O 스트림을 사용
: 키보드, 파일 디스크, 메모리 등
