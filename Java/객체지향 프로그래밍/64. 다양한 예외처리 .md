
> 예외 처리 미루기

+ throws를 사용하여 예외처리 미루기 (어디로 미뤄?? 이 메소드를 호출한 곳에서 예외처리를 하겠다.)

+ try{} 블록으로 예외를 처리하지 않고, 메서드 선언부에 throws를 추가

+ 예외가 발생한 메서드에서 예외처리를 하지 않고 이 메서드를 호출한 곳에서 예외처리를 한다는 의미

+ main() 에서 throws를 사용하면 가상머신에서 처리됨
